             Structural Comparison of java files. Comparison should be on raw files, i.e., java files should not be compiled. Creating AST trees using JDT plugin APIs Comparing these AST trees against each other and listing out the differences. AST tree are mainly compared for differences in 3 structural property types viz., package, imports and types. Difference object should be constructed based on the comparison.
